PROJECT OVERVIEW
----------------------------------------------------------

Blood Donor Finder is a modern mobile application UI/UX
design created using Figma. The application is designed
to connect blood donors with recipients quickly during
emergencies through an intuitive and user-friendly
interface.

The design focuses on simplicity, accessibility, and
efficient navigation to provide a seamless user experience.
