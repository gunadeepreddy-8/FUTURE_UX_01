## 📌 Project Overview

Blood Donor Finder is a mobile application UI/UX design created in **Figma** to simplify blood donation and emergency blood requests.

The application allows users to:

- Register as a blood donor
- Login securely
- Search nearby blood donors
- Filter donors by blood group
- View donor profiles
- Request blood during emergencies
- Contact donors instantly
- Receive request confirmation
